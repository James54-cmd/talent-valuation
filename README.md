# Talent Valuation Platform

A professional talent valuation and assessment platform built with React and TypeScript.

## Getting Started

Install dependencies:
```bash
npm install
```

Start the development server:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```

## Technology Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- ESLint
- Vitest for testing
- Playwright for e2e testing
