# SalarioPH

Philippine salary benchmarking and city comparison for job seekers, recruiters, and hiring teams.

Live app:

```txt
https://talent-valuation.vercel.app/
```

## What It Does

SalarioPH is a Philippines-first salary guide built around how compensation is actually discussed locally:

- monthly salary first
- annual context second
- explicit 13th month visibility
- Philippine city context
- benchmark ranges instead of fake exact salary promises

Current product areas:

- `Estimate salary`
  - personalized salary range based on role, experience, location, industry, work setup, company type, and specialization
- `Compare by city`
  - researched city comparisons for roles where the benchmark evidence is strong enough to surface
- `PH benchmarks`
  - benchmark coverage and confidence context behind the estimator
- `How it works`
  - methodology and trust framing for the product

## Architecture Notes

- `src/app` stays thin and handles routes, metadata, and API boundaries.
- `src/features/salary-estimator` owns the product logic, UI composition, benchmark behavior, and methodology-facing experience.
- `src/components/ui` contains reusable shadcn/ui primitives.
- `src/components/common` and `src/components/layout` contain shared presentation and layout building blocks.
- `src/lib/db` contains Supabase client setup for browser, server, and admin access.
- `supabase/` contains migrations and seed scaffolding for benchmark data.

## Product Notes

- The app is intentionally Philippines-first, not a global salary calculator adapted later.
- Monthly gross salary is the primary output.
- 13th month pay is shown explicitly.
- City comparison data is currently research-backed and selectively surfaced where confidence is strong enough.
- The UI uses warm editorial-fintech styling rather than default SaaS dashboard patterns.
